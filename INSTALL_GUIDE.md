# 記事シェア Chrome拡張機能 インストールガイド

## 📥 インストール手順

### ステップ1: リポジトリをダウンロード

以下のいずれかの方法でリポジトリを取得してください。

#### 方法A: Gitを使用（推奨）

```bash
git clone https://github.com/bassie0303/news.git
cd news
```

#### 方法B: ZIPファイルをダウンロード

1. https://github.com/bassie0303/news にアクセス
2. 緑色の「Code」ボタンをクリック
3. 「Download ZIP」を選択
4. ダウンロードしたZIPファイルを解凍

### ステップ2: Chrome拡張機能として読み込む

1. **Chromeブラウザを開く**

2. **拡張機能管理ページを開く**
   - アドレスバーに `chrome://extensions/` と入力してEnterキーを押す
   - または、メニュー（⋮）→「拡張機能」→「拡張機能を管理」

3. **デベロッパーモードを有効化**
   - ページ右上の「デベロッパーモード」トグルをONにする

4. **拡張機能を読み込む**
   - 「パッケージ化されていない拡張機能を読み込む」ボタンをクリック
   - ダウンロード/クローンしたフォルダ（`news`）を選択
   - 「フォルダーの選択」をクリック

5. **インストール完了**
   - 拡張機能一覧に「記事シェア拡張機能」が表示されます
   - ツールバーに青い円のアイコンが表示されます

## 🚀 使い方

### 基本的な使用方法

1. **記事ページを開く**
   - 投稿したいニュース記事やブログ記事のページをChromeで開きます

2. **拡張機能を起動**
   - ツールバーの青い円アイコンをクリック
   - ポップアップが表示されます

3. **記事情報を確認**
   - タイトル、URL、自動抽出された抜粋が表示されます

4. **SNSに投稿**
   - 投稿方法を選択：
     - 🚀 **すべてに一括投稿** → Facebook、Threads、Xの3つに同時に投稿画面が開きます（おすすめ！）
     - 🔵 **Facebookに投稿** → Facebook投稿画面のみが開きます
     - ⚫ **Threadsに投稿** → Threads投稿画面のみが開きます
     - ⚫ **Xに投稿** → X投稿画面のみが開きます

5. **投稿を確認して送信**
   - 自動入力された内容を確認
   - 必要に応じて編集
   - 投稿ボタンを押して完了

## 📱 各SNSの設定

### Facebook
- **事前準備**: Facebookにログインしておく
- **文字数制限**: 1000文字以内に自動調整
- **投稿フォーマット**:
  ```
  記事タイトル

  「抜粋された記事の一部」

  記事URL
  ```

### Threads
- **事前準備**: Threadsにログインしておく
- **文字数制限**: 500文字以内に自動調整
- **投稿フォーマット**: Facebookと同じ

### X (旧Twitter)
- **事前準備**: Xにログインしておく
- **文字数制限**: 140文字以内に自動調整
- **投稿フォーマット**:
  ```
  記事タイトル
  「抜粋」
  URL
  ```

## ⚙️ 抜粋の仕組み

拡張機能は以下の基準で記事から重要な情報を自動抽出します：

✅ **優先的に抽出される内容**:
- 📊 数値データ（統計、パーセンテージなど）
- 💬 引用符で囲まれた発言
- 👤 人名や固有名詞
- 🔑 重要キーワード（「発表」「発見」「開発」など）

この仕組みにより、記事の核心部分を効率的に抽出できます。

## 🔧 トラブルシューティング

### 問題1: 記事情報が取得できない

**原因**: ページが完全に読み込まれていない

**解決策**:
- ページの読み込みが完全に終わってから拡張機能を起動
- ページを再読み込み（F5）してから再試行

### 問題2: 投稿ボックスにテキストが入力されない

**原因**: SNSのUIが変更されている可能性

**解決策**:
- アラートが表示された場合、投稿内容をコピーして手動で貼り付け
- 拡張機能を最新版に更新

### 問題3: ログインを求められる

**原因**: SNSにログインしていない

**解決策**:
- 各SNSに事前にログイン
- Chromeのログイン状態は保持されます

### 問題4: 拡張機能が表示されない

**原因**: インストールが正しく完了していない

**解決策**:
1. `chrome://extensions/` を開く
2. 「記事シェア拡張機能」が有効になっているか確認
3. 無効の場合、トグルをONにする
4. それでも表示されない場合、再インストール

## 🔄 更新方法

### Gitを使用している場合

```bash
cd news
git pull origin main
```

その後、`chrome://extensions/` で拡張機能の「再読み込み」ボタンをクリック

### ZIPでダウンロードした場合

1. 最新のZIPファイルをダウンロード
2. 解凍
3. `chrome://extensions/` で古いバージョンを削除
4. 新しいフォルダを読み込む
