# 記事シェア拡張機能 使用方法ガイド

## 🚀 基本的な使い方

### ステップ1: 記事を開く
投稿したいニュース記事やブログ記事をChromeで開きます。

### ステップ2: 拡張機能を起動
ツールバーの青い円アイコンをクリックします。

### ステップ3: 投稿方法を選択

#### 🎯 おすすめ: 一括投稿
**「🚀 すべてに一括投稿」ボタンをクリック**

- Facebook、Threads、Xの3つのタブが自動的に開きます
- 投稿内容はクリップボードにコピーされます
- 各タブで投稿内容を確認して投稿ボタンを押してください

#### 個別投稿
各SNSのボタンを個別にクリックすることもできます。

## 📝 投稿の流れ

### Facebook / Threads の場合

1. **拡張機能でボタンをクリック**
   - 該当するSNSのタブが開きます
   - 投稿内容はクリップボードにコピーされます

2. **投稿ボックスをクリック**
   - 自動入力が失敗した場合、投稿ボックスをクリックしてください

3. **貼り付け**
   - `Ctrl+V`（Windows）または `Cmd+V`（Mac）で貼り付け
   - または右クリック→「貼り付け」

4. **投稿ボタンを押す**
   - 内容を確認して投稿ボタンをクリック

### X の場合

1. **拡張機能でボタンをクリック**
   - Xの投稿画面が開きます
   - 投稿内容が自動的に入力されます

2. **投稿ボタンを確認**
   - 自動的に投稿ボタンがクリックされます
   - されない場合は手動で「ポスト」ボタンをクリック

## 💡 ヒント

### クリップボード機能
すべての投稿内容は自動的にクリップボードにコピーされます。
- 自動入力が失敗しても、`Ctrl+V`で簡単に貼り付けられます
- 複数のSNSに投稿する場合も便利です

### 一括投稿の効率的な使い方

1. **「🚀 すべてに一括投稿」をクリック**
2. **3つのタブが開きます**
   - Facebook（バックグラウンド）
   - Threads（バックグラウンド）
   - X（アクティブタブ）

3. **Xから投稿開始**
   - アクティブなXタブで投稿を確認
   - 投稿ボタンをクリック（または自動クリック済み）

4. **Facebookタブに移動**
   - タブを切り替え
   - 投稿ボックスをクリックして`Ctrl+V`で貼り付け
   - 投稿

5. **Threadsタブに移動**
   - タブを切り替え
   - 投稿ボックスをクリックして`Ctrl+V`で貼り付け
   - 投稿

## ⚠️ トラブルシューティング

### 自動入力が動作しない場合

**原因**: SNSのUIが変更されている可能性があります

**解決策**:
1. 投稿内容はクリップボードにコピーされています
2. 投稿ボックスをクリック
3. `Ctrl+V`（または`Cmd+V`）で貼り付け
4. 投稿ボタンをクリック

### 投稿ボタンが見つからない

**解決策**:
- ページが完全に読み込まれるまで待つ
- ページを再読み込み（F5）
- 手動で投稿ボックスに貼り付けて投稿

### ログインを求められる

**解決策**:
- 各SNSに事前にログインしておく
- ログイン状態は保持されます

### 記事情報が取得できない

**解決策**:
- ページが完全に読み込まれてから拡張機能を起動
- ページを再読み込み（F5）してから再試行

## 📋 各SNSの文字数制限

| SNS | 文字数制限 | 調整方法 |
|-----|-----------|---------|
| **Facebook** | 1000文字 | 自動調整（抜粋を短縮） |
| **Threads** | 500文字 | 自動調整（抜粋を短縮） |
| **X** | 140文字 | 自動調整（抜粋を大幅短縮） |

**注意**: タイトルとURLは省略されません

## 🎨 投稿フォーマット

すべてのSNSで以下のフォーマットが使用されます：

```
記事タイトル

「抜粋された記事の一部」

記事URL
```

## 🔄 更新方法

拡張機能を更新した場合：

1. `chrome://extensions/` を開く
2. 「記事シェア拡張機能」の「再読み込み」ボタンをクリック
3. 開いているSNSのタブをリロード
