# 記事シェア Chrome拡張機能

ネット記事を自動的に抜粋してFacebook、Threads、Xに投稿するChrome拡張機能です。

## 機能

- **自動抜粋**: 記事から重要な情報（人名、発言、数値など）を自動的に抽出
- **複数SNS対応**: Facebook、Threads、Xに対応
- **文字数制限対応**: 各SNSの文字数制限に合わせて自動調整
  - Facebook: 1000文字以内
  - Threads: 500文字以内
  - X: 140文字以内
- **引用符付与**: 抜粋部分に自動的に引用符を付与

## インストール方法

### 1. リポジトリをクローン

```bash
git clone https://github.com/bassie0303/news.git
cd news
```

### 2. Chrome拡張機能として読み込み

1. Chromeブラウザを開く
2. アドレスバーに `chrome://extensions/` と入力してEnter
3. 右上の「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. クローンしたフォルダを選択

## 使い方

### 基本的な使い方

1. 投稿したい記事のページをChromeで開く
2. ブラウザのツールバーにある拡張機能アイコン（青い円）をクリック
3. ポップアップに記事のタイトル、URL、抜粋が表示される
4. 投稿方法を選択
   - **🚀 すべてに一括投稿**: Facebook、Threads、Xの3つに同時に投稿画面を開く（おすすめ！）
   - **Facebookに投稿**: Facebook投稿画面のみを開く
   - **Threadsに投稿**: Threads投稿画面のみを開く
   - **Xに投稿**: X投稿画面のみを開く
5. 各タブで内容を確認して投稿ボタンを押す

### 投稿フォーマット

各SNSへの投稿は以下のフォーマットで生成されます：

```
記事タイトル

「抜粋された記事の一部」

記事URL
```

### 抜粋の仕組み

拡張機能は以下の基準で重要な情報を抽出します：

- **数値データ**: 統計や具体的な数字を含む文
- **発言**: 引用符で囲まれた発言
- **人名**: 固有名詞や人物名
- **重要キーワード**: 「発表」「発見」「開発」などの重要な動詞を含む文

これらの要素を含む文を優先的に抽出し、各SNSの文字数制限内に収まるように調整します。

## 技術仕様

- **Manifest Version**: 3
- **必要な権限**:
  - `activeTab`: 現在のタブの情報を取得
  - `scripting`: ページにスクリプトを注入
  - `storage`: 設定の保存（将来の機能拡張用）
- **対応ホスト**:
  - Facebook
  - Threads
  - X (Twitter)

## トラブルシューティング

### 記事情報が取得できない場合

- ページが完全に読み込まれてから拡張機能を起動してください
- 一部の動的なWebサイトでは正しく動作しない場合があります

### 投稿ボックスにテキストが入力されない場合

- SNSのUIが変更されている可能性があります
- その場合は、表示されたアラートから投稿内容をコピーして手動で貼り付けてください

### ログインが必要な場合

- 各SNSに事前にログインしておいてください
- Chromeのログイン状態は保持されます

## 開発者向け情報

### ローカル開発

1. コードを編集
2. `chrome://extensions/` で「再読み込み」ボタンをクリック
3. テスト

### デバッグ

- ポップアップのデバッグ: ポップアップを右クリック → 「検証」
- コンテンツスクリプトのデバッグ: ページを右クリック → 「検証」→ Consoleタブ
- バックグラウンドスクリプトのデバッグ: `chrome://extensions/` → 「Service Worker」をクリック
